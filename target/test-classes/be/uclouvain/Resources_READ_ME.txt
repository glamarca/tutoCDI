Pour créer les fichiers resources.properties (bundle) : 
- Créer le répertoire du project
- Dans ce repertoire créer le dossier web
- Dans ce répertoire web créer le repertoire bundle
- Dans ce repertoire bundle créer les fichiers properties : 
		- Resources.properties est le fichier par defaut
		- Resources_en.properties serait le fichier en anglais, Resources_es.properties , le fichier en espagnol, ....
